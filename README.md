## 🚀 Deployment

### Automated Pipeline

1. **Push changes** to the `main` branch  
2. **Netlify auto-builds** and deploys the site  
3. **Domain propagates** via Cloudflare DNS → `kypriatechnologies.org`  
4. **Monitoring activates** through Cloudflare Analytics  

### Manual Deployment

```bash
# Clone the repository
git clone https://github.com/Kypria-LLC/kypria-technologies.git
cd kypria-technologies

# Serve locally for testing
python3 -m http.server 8000
# OR
npx http-server -p 8000

# Visit http://localhost:8000
```
